# Images pour Chapitre 1 : Ondes Mécaniques Progressives

## Images requises pour cette leçon

### 1. onde_transversale.png
**Description** : Schéma d'une onde transversale sur une corde
- Montre une corde avec une perturbation perpendiculaire à la direction de propagation
- Flèches indiquant la direction de propagation (horizontale) et la direction de la perturbation (verticale)
- Légendes en français et arabe

**Utilisation** : Phase d'explication, quand le tuteur explique la différence entre ondes transversales et longitudinales

**Trigger IA** : "regarde ce schéma", "observe cette image d'onde transversale"

---

### 2. onde_longitudinale.png
**Description** : Schéma d'une onde longitudinale (ressort comprimé)
- Montre un ressort avec zones de compression et dilatation
- Flèches indiquant que la perturbation est parallèle à la propagation
- Légendes bilingues

**Utilisation** : Phase d'explication, comparaison avec onde transversale

**Trigger IA** : "voici une onde longitudinale", "regarde comment le son se propage"

---

### 3. pierre_eau_animation.gif
**Description** : Animation d'une pierre tombant dans l'eau créant des cercles concentriques
- Montre la propagation des ondes circulaires
- Optionnel : bouchon flottant qui monte/descend sans se déplacer latéralement

**Utilisation** : Phase d'exploration, scénario "La pierre dans l'eau"

**Trigger IA** : "imagine que tu jettes une pierre dans l'eau", "observe ce qui se passe"

---

### 4. celerite_schema.svg
**Description** : Schéma explicatif de la célérité
- Deux points A et B sur une corde
- Distance d entre les deux points
- Temps τ (tau) pour que l'onde aille de A à B
- Formule v = d/τ avec légendes

**Utilisation** : Phase d'explication, quand le tuteur explique la formule de célérité

**Trigger IA** : "voici comment calculer la célérité", "regarde ce schéma de calcul"

---

## Comment obtenir ces images

### Option 1 : Créer avec des outils en ligne (gratuit)
- **Canva** : https://www.canva.com/ (templates scientifiques)
- **Excalidraw** : https://excalidraw.com/ (dessins simples)
- **PhET Simulations** : https://phet.colorado.edu/ (captures d'écran)

### Option 2 : Banques d'images libres
- **Wikimedia Commons** : https://commons.wikimedia.org/
- **Pixabay** : https://pixabay.com/
- Rechercher : "wave transverse", "mechanical wave", "longitudinal wave"

### Option 3 : Générer avec IA
- **DALL-E / Midjourney** : Prompts comme "educational diagram of transverse wave on a rope, labeled in French and Arabic"

## Format recommandé
- **Format** : PNG (transparence) ou SVG (vectoriel)
- **Résolution** : 1200x800px minimum
- **Poids** : < 500KB par image
- **Texte** : Bilingue français/arabe
- **Style** : Clair, éducatif, couleurs contrastées
